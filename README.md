# LogObserver
指定した.txtまたは.logを監視し、新規行をコンソールに出力するツールです  

## How to use
java -jar LogObserver.jar 対象ファイル 出力間隔  

対象ファイル:カレントディレクトリにあるファイル名、またはファイルの絶対パスを指定  
出力間隔:コンソールに出力する間隔 未指定または数値以外を指定した場合1秒  

終了する場合はコンソールでCtrl + c を押下してください
